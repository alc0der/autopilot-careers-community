
## /cover

Draft a <new_job_cover_letter>.

Follow these steps:
1. Read the job description
2. Read the resume
3. Read the cover letter draft (if any)
4. Read the critique (if any)
5. Draft a new cover letter following these guidelines:
  - Avoid repeating what is already in the resume
  - Avoid repeating what is already in the job description
  - Paraphrase ideas
  - Make it short and brief
