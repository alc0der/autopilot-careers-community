# Generic Resume

Create a generic resume targeting a role (and optionally a company) without a specific job posting. Use the `write-resume` skill's generic workflow to build a resume from the user's career data, optimized for the target role.

## Usage

```
/generic-resume $ARGUMENTS
```

Pass the role title and optional company to the skill's `/generic` flow.
