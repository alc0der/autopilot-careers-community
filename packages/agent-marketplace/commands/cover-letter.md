# Cover Letter

Draft a cover letter tailored to a job application. Use the `write-resume` skill's cover letter workflow, drawing from the user's career data and the job description to craft a compelling letter.

## Usage

```
/cover-letter $ARGUMENTS
```

If a job description has already been fetched in this session, use it. Otherwise, fetch it first using the provided URL.
