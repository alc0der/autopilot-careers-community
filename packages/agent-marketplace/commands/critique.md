# Critique

Critique an existing resume or cover letter against the job description. Use the `write-resume` skill's critique workflow to identify strengths, gaps, and specific improvements.

## Usage

```
/critique $ARGUMENTS
```

If no file is specified, critique the most recently generated resume. If a job description is available, evaluate fit against it.
